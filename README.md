## 🚀 How to Use

1. Clone this repository:

   ```bash
   git clone https://github.com/yourusername/shell-scripting-projects.git
   cd shell-scripting-projects
   ```

2. Navigate to the script you want to run:

   ```bash
   cd folder-name
   ```

3. Give it execute permission and run:

   ```bash
   chmod +x script.sh
   ./script.sh
   ```
